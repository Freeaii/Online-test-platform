#oec
---

开发环境
* node.js 16
* vscode
* vscode 插件 

# 下载
* 命令行执行
``` shell 
  git clone https://e.coding.net/mlweb/online-exercise/oec.git 
```

* 用vscode打开文件夹 oec
* 在vscode的终端运行下面命令进行库安装
```shell
  npm install
```
* 安装完成后运行，在确认后端已运行基础上运行前端
```shell
  npm run dev
```
* 用浏览器打开
> http://localhost:3000
